# enumUnauth
#在平常分析的固件的过程中，会有一些web目录中的页面存在未授权访问的情况，如NetGear DGN 1000中的currentsetting.htm,写了一个小脚本，方便的枚举web目录中的未授权访问
#安装
#sudo chmod 777 ./install.sh
#sh install.sh

#使用
#binwalk解包并模拟
#cd进入固件的web目录
